# v1.6.3-o7: C++ 네이티브 연산 가속(AVX2 SIMD & 멀티스레딩) 및 인간 시각 감도 기반 적응형 색상 압축 최적화 릴리즈

## 릴리즈 개요 (Release Summary)

**v1.6.3-o7** 릴리즈는 `MecchaCamouflage` 도색 시스템의 네이티브 연산 커널 성능을 극대화하고, 도색 품질 손실 없이 획수를 획기적으로 줄이기 위한 **C++ 네이티브 연산 가속(AVX2 SIMD & 멀티스레딩)** 및 **인간 시각 감도 기반 적응형 색상 압축 알고리즘**을 공식 도입한 정식 릴리즈 버전입니다.

전처리 및 도색 계획 수립 과정의 하위 4대 구간(`pose`, `capture`, `sample`, `plan`) 소요 시간을 0.1ms 정밀도로 분해 측정하여 보여주는 **고정밀 세부 성능 측정 로거(High-Precision Telemetry Logger)**가 기본 탑재되어 있습니다.

---

## 주요 변경 사항 및 기술 명세 (Key Improvements & Features)

### 1. 로직 및 네이티브 연산 가속 (Logic & Native Compute Acceleration)

- **1-1. AVX2 SIMD 벡터화 (AVX2 SIMD Vectorization)**
  - **기술 상세**: C++ 네이티브 `color_distance_squared` 핵심 커널에 Intel/AMD CPU 256비트 YMM 레지스터 명령어 (`_mm256_fmadd_pd`, `_mm256_mul_pd`, `_mm256_sub_pd`)를 적용하여 2D UV 공간 대량 픽셀의 시각적 색상 거리 연산을 하드웨어 레벨에서 벡터화 병렬 수행합니다.
  - **기대 효과**: 도색 계획 수립 중 발생하는 대량의 픽셀 간 색상 오차 거리 계산 연산 오버헤드를 하드웨어 레벨에서 수직 감소시켜 CPU 연산 병목을 전면 해소합니다.

- **1-2. 표준 C++ Zero-Dependency 멀티스레드 공간 탐색 (Multithreaded Spatial Search)**
  - **기술 상세**: MSVC `/openmp` 빌드 시 발생하는 외부 DLL 의존성(`VCOMP140.DLL`) 주입 오류를 차단하기 위해, 순수 C++17 표준 `<future>` / `<thread>` (`std::async`) 기반의 척크 병렬화 탐색 커널을 구현하여 8,000+ 후보 브러시 배율 검증 탐색을 멀티코어 CPU 전 스레드로 병렬 처리합니다.
  - **기대 효과**: 적응형 계획 수립(`plan`) 연산 속도를 복잡도가 높은 맵에서 **최대 27.3% 빠른 수직 가속**을 달성하며, 외부 DLL 종속성 없는 단일 파일 바이너리 격리 안전성을 보장합니다.

- **1-3. 2D UV 공간 동적 격자 분할 (Spatial Grid Partitioning)**
  - **기술 상세**: 2D UV 표면 샘플 개수($N$)에 따라 $128 \times 128 \sim 512 \times 512$ 셀 격자를 동적으로 분할 구축하고, 연속 메모리 바운딩 헤더(393KB~6.2MB)를 적용하여 CPU L2/L3 캐시 상주 효율성을 극대화합니다.
  - **기대 효과**: 이웃 픽셀 공간 검색 시간 복잡도를 $O(N^2)$에서 $O(1)$로 단축시키고 탐색 중 CPU 캐시 미스(Cache Miss)를 차단하여 초고밀도 메쉬에서도 순간 프레임 드랍 없이 도색 계획을 수립합니다.

- **1-4. 고정밀 세부 성능 측정 로거 (High-Precision Telemetry Logger)**
  - **기술 상세**: 도색 전처리 과정의 4대 핵심 하위 구간(`pose`: 포즈 연산, `capture`: SceneCapture2D GPU 픽셀 읽기, `sample`: 2D UV 샘플링, `plan`: C++ 적응형 계획 수립) 소요 시간을 0.1ms 정밀도로 분해 측정하여 콘솔 로그 및 프로그래밍 인터페이스로 표출합니다.
  - **기대 효과**: 시스템의 병목 지점을 0.1ms 단위로 정밀하게 추적 및 디버깅 가능하며, 프로덕션 환경에서의 실시간 렌더링 성능을 명확하게 입증할 수 있습니다.

### 2. 도색 및 시각 감도 기반 알고리즘 최적화 (Paint Optimizations)

- **2-1. 인간 시각 감도 기반 색상 오차 공식 (Weighted Perceptual Color Metric)**
  - **기술 상세**: 단순 유클리드 RGB 거리 대신 인간 안구의 파장별 명암 민감도 파장(초록 58.7%, 빨강 29.9%, 파랑 11.4%)을 반영한 가중치 오차 공식($\Delta C^2 = 0.299 \Delta R^2 + 0.587 \Delta G^2 + 0.114 \Delta B^2$)을 적용합니다.
  - **기대 효과**: 인간의 눈이 인지하기 어려운 미세한 파랑/어둠 파장의 색상 차이로 인해 불필요하게 붓질이 분할되는 현상을 방지하여 도색 품질 손실 없이 획수를 획기적으로 줄입니다.

- **2-2. Gradient-Aware 동적 가변 오차 (Gradient-Aware Dynamic Tolerance)**
  - **기술 상세**: 메시 텍스처 표면의 국소 2D 색상 변화량(Gradient)을 계산하여, 평면 및 단색 구역에서 브러시 반경 오차 범위를 최대 8.0배까지 동적으로 확장하여 스트로크 수를 압축합니다.
  - **기대 효과**: 넓은 평면 및 단색 부위의 도색 획수를 최대 8.0배까지 대량 통합하여 전체 도색 시간을 획기적으로 단축시킵니다 (콘크리트 벽 기준 도색 완수 시간 8초 $\rightarrow$ 7초로 단축).

- **2-3. 암부/음영 구역 초고압축 (Luminance Shadow Masking)**
  - **기술 상세**: 메시 픽셀의 휘도($Y < 0.25$)를 실시간 측정하여 어두운 그림자 구역에서 인간의 색 구별 능력이 크게 저하되는 특성을 반영해 색상 압축 허용치를 상향 스케일링합니다.
  - **기대 효과**: 그림자 및 음영 부위의 불필요한 미세 잔여 스트로크 조각을 깔끔하게 흡수 소거하여 획수 절감 및 도색 완료 안정성을 대폭 향상시킵니다.

- **2-4. 기체 관절/프레임 구역 경계선 보존 (Joint Boundary Sharpness Retention)**
  - **기술 상세**: `same_payload` (부품 단위 UV 아일랜드 및 Normal 구역 비교) 검사를 수행하고, 패널 라인 및 고주파 에지 부근 픽셀(`center.safe == false`)의 브러시 반경 확장을 차단하여 1.0x 정밀 세밀 브러시 크기로 클램핑(Clamp)시킵니다.
  - **기대 효과**: 메카닉 기체의 부품 장갑판 경계선, 관절, 패널 라인 부근의 위장 도색 경계가 뭉개지거나 번지는 현상(Bleeding/Blur)을 원천 차단하여, 넓은 평면은 빠르고 넓게 도색하면서도 경계선 윤곽의 입체감과 선명도를 100% 보존합니다.

## 핵심 검증 및 바이너리 안내

- **전체 단위 테스트**: 76개 C# 및 C++ Native 단위 테스트 **100% PASS** (0 Errors, 0 Warnings).
- **외부 DLL 격리 검증**: MSVC `/openmp` 제거 및 `std::async` 채택으로 `VCOMP140.DLL` 종속성 무오류 보장.
- **실행 파일 위치**: `.build/bin/meccha-camouflage.exe` (단일 포터블 바이너리).
